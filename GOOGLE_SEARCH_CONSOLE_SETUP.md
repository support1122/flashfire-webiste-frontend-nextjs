1. Go to **Sitemaps** in left sidebar
2. In "Add a new sitemap" field, enter: `sitemap.xml`
3. Click **"Submit"**

### What to Expect:

## ‚úÖ Step 3: Request Re-indexing of Important Pages

### Priority Pages to Re-index:
1. **Homepage:** `https://www.flashfirejobs.com/`
2. **Pricing:** `https://www.flashfirejobs.com/pricing`
3. **Blogs:** `https://www.flashfirejobs.com/blogs`

### Steps:
1. Go to **URL Inspection** tool (top search bar)

**Note:** Don't request indexing for all pages at once. Start with 5-10 most important pages.

## ‚úÖ Step 4: Monitor Coverage Report

### Check for Issues:
1. Go to **Coverage** in left sidebar
2. Check for:
   - ‚ùå **Errors** - Pages that can't be indexed
   - ‚ö†Ô∏è **Warnings** - Pages with issues but still indexed
   - ‚úÖ **Valid** - Pages indexed successfully
   - üìÑ **Excluded** - Pages intentionally excluded

### What to Look For:
- **404 errors** - Broken links (should be minimal if URLs match old site)
- **Redirect errors** - If you changed any URLs
- **Server errors** - Backend issues
- **Robots.txt blocking** - Check if robots.txt is blocking important pages

## ‚úÖ Step 5: Verify robots.txt

### Check robots.txt is Accessible:
1. Go to **Settings** ‚Üí **robots.txt Tester**
2. Enter: `https://www.flashfirejobs.com/robots.txt`
3. Verify it shows:
   ```

   Allow: /
   Sitemap: https://www.flashfirejobs.com/sitemap.xml
   ```

### Test Important URLs:
- Test your homepage URL
- Test a few other important pages
- Ensure they're not blocked

1. Go to **Performance** in left sidebar
2. Monitor:
   - **Impressions** - Should remain stable or increase
   - **Clicks** - Should remain stable
   - **Average Position** - Should not drop significantly
   - **CTR** - Click-through rate

### What to Watch For:
- üìâ **Drop in impressions/clicks** - May indicate indexing issues
- üìâ **Drop in average position** - May need time to stabilize
- ‚úÖ **Stable or improving metrics** - Good sign!

## ‚úÖ Step 8: Set Up URL Parameters (If Needed)

### If you use UTM parameters:
1. Go to **Settings** ‚Üí **URL Parameters**
2. Add parameters you want Google to ignore:
   - `utm_source`
   - `utm_medium`
   - `utm_campaign`
   - `utm_content`
   - `utm_term`

**Note:** This prevents duplicate content issues from UTM parameters.

## üÜò Troubleshooting

### Sitemap Not Found:
- Wait 24-48 hours after deployment
- Verify sitemap is accessible: `https://www.flashfirejobs.com/sitemap.xml`
- Check Next.js build logs

### Pages Not Indexing:
- Check robots.txt isn't blocking
- Verify pages return 200 status code
- Request indexing manually
- Check for noindex tags (shouldn't have any)

### Rankings Dropping:
- Normal during first 2-4 weeks
- Ensure all important pages are indexed
- Check for duplicate content issues
- Verify canonical URLs are correct
